# fraud-detection-audit.NAU-audit-acca--data-practice
基于python及CDA2级框架的内部审计数据分析实践--以信用卡欺诈风险识别为列
南京审计大学大一学生基于python的审计数据分析实战项目,包含EDA，WOE分箱，逻辑回归及学审计学(acca)专业大一实践中作品，于acca基础阶段及CDA数据分析师认证2级备考期间完成项目.
数据集来源于kaggle公开已去敏数据集(Credit Card Fraud Detection)，数据集URL:https://www.kaggle.com/datasets/kaushalnandania/credit-card-fraud-detection，相关算法及逻辑思路借鉴auro15和 Kaushal Nandaniya2位分享.
结合CDA数据分析技术栈和ACCA知识点相关嵌入,希望体现个人作为审计专业学生兼ACCA方向班的国际化思维，结合CDA数字化技能，致力于智能审计及大数据审计发展方向，体现个人大一的成长轨迹。
